About Orbigator

Orbigator is a precision orbital mechanics demonstrator using Dynamixel motors to simulate satellite orbits and track real satellites using SGP4 propagation.

Features:

  • Real-time satellite tracking (SGP4)
  • Custom orbital parameter simulation
  • Elliptical orbit support
  • Web-based remote control
